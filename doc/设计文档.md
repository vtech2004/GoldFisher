# 黄金钓鱼 - 游戏设计文档

## 1. 游戏概述

### 1.1 游戏名称
**黄金钓鱼（Gold Fisher）**

### 1.2 游戏类型
休闲益智类单机/联网小游戏，基于经典的"黄金矿工"玩法换皮为钓鱼主题。

### 1.3 游戏简介
玩家扮演一名钓鱼高手，在一片富饶的海域中通过控制鱼钩（钟摆式摆动）来钓起水下的各类鱼类、珍宝和道具。每个关卡设有目标金额和限时，玩家需在规定时间内钓到足够价值的物品以达成目标并进入下一关。游戏中还包含陷阱物品和特殊道具，增加策略性和趣味性。

### 1.4 玩法核心
- **钟摆钩爪机制**：鱼钩在屏幕上方左右摆动，玩家点击/按键释放鱼钩
- **抓取机制**：鱼钩下落碰到物品后抓住物品并收回
- **收回速度与重量相关**：物品越重，收回速度越慢
- **计分机制**：根据钓到的物品价值累加分数
- **关卡递进**：关卡难度逐渐增加，目标金额提升

### 1.5 目标平台
- PC（Windows）
- 移动端（可选扩展）

### 1.6 开发引擎
- Unity 2022.3.34f1c1

## 2. 核心玩法

### 2.1 钟摆钩爪机制

游戏开始时，鱼钩悬挂在画面顶部中央，以固定支点为轴左右摆动（类似钟摆运动）。

- **摆动角度**：左右各75度（共150度摆动范围）
- **摆动周期**：约2秒一个来回（关卡递进可加快）
- **释放时机**：玩家通过点击屏幕或按下空格键释放鱼钩
- **释放后**：鱼钩沿当前摆动方向直线下降

### 2.2 抓取机制

鱼钩释放后垂直下落，遇到第一个物品即触发抓取：

- **抓取判定**：鱼钩尖端与物品发生碰撞时抓取
- **抓取后**：物品附着在鱼钩上，鱼钩开始收回
- **未抓到**：鱼钩到达画面底部（或设定深度）后空手收回
- **收回速度公式**：
  ```
  收回速度 = 基础速度 / (1 + 物品重量 * 重量系数)
  ```
  - 基础速度：800 像素/秒
  - 重量系数：0.15
  - 重量越大，收回越慢

### 2.3 收回机制

鱼钩抓住物品后，沿释放时的反向路径收回至顶部支点：

- 收回过程中不可再次释放
- 收回完成后，物品价值计入本关分数
- 物品被收入玩家"鱼篓"

### 2.4 计分机制

- 每次成功钓起物品，将其价值累加到本关得分
- 本关得分实时显示在UI上
- 关卡结束时，若得分 ≥ 目标金额，则通关；否则失败
- 通关后剩余时间可转换为 bonus 奖励（剩余秒数 × 10）

### 2.5 关卡失败条件
- 时间耗尽且未达到目标金额 → 失败
- 失败后可重试本关

## 3. 物品设计表

水下物品共分为四大类：**鱼类**、**珍宝**、**陷阱**、**道具**。

| 编号 | 名称 | 类型 | 价值(分) | 重量 | 出现概率 | 说明 |

| 1 | 小鱼 | 鱼类 | 50 | 1.0 | 高 | 常见基础鱼，价值低但易获取 |
| 2 | 中鱼 | 鱼类 | 100 | 2.0 | 中 | 价值与重量均衡 |

### 3.1 物品设计说明

#### 小鱼
- **定位**：基础填充物，保证玩家不会空手
- **策略**：剩余时间不足时优先钓小鱼刷分

#### 中鱼 / 大鱼 / 鲨鱼
- **定位**：核心得分来源，重量与价值递增
- **策略**：鲨鱼价值最高但收回极慢，需权衡时间成本

#### 水母 / 螃蟹
- **定位**：移动型物品，增加操作难度
- **螃蟹**会横向左右移动，水母会上下浮动

#### 珍珠
- **定位**：性价比之王（价值/重量比最高）
- **设计意图**：鼓励玩家精准操作去钓小目标

#### 宝箱
- **定位**：稀有大奖
- **设计意图**：钓到宝箱可扭转局势

#### 金币袋
- **定位**：中等珍宝，平衡选择

#### 旧靴子
- **定位**：陷阱物品
- **设计意图**：考验玩家精准度，避免钓到无价值物品浪费时间

#### 炸弹
- **定位**：策略道具
- **使用方式**：钓到炸弹后会自动存入道具栏，玩家可在鱼钩收回过程中点击"炸毁"按钮，丢弃当前钩上的物品（用于丢弃旧靴子等陷阱或过重的鲨鱼），鱼钩立即快速收回
- **设计意图**：增加策略深度，给玩家补救机会

## 5. 计分规则

### 5.1 基础计分
- 每钓起一个物品，将物品价值加入本关得分
- 炸弹本身价值为0，不计分

### 5.2 时间奖励（Bonus）
通关后，剩余时间转换为奖励分数：
```
时间奖励 = 剩余秒数 × 10
```

### 5.3 总分计算
```
关卡总分 = 物品得分合计 + 时间奖励
```

### 5.4 累计积分
- 玩家累计积分 = 所有已通关关卡总分之和
- 累计积分用于全服排行榜

### 5.5 计分示例
> 玩家在第1关钓到：小鱼×3 + 中鱼×2 + 金币袋×1 = 50×3 + 100×2 + 200 = 550分
> 关卡目标500，达成。剩余时间15秒。
> 时间奖励 = 15 × 10 = 150
> 第1关总分 = 550 + 150 = 700分

## 6. 道具系统

### 6.1 炸弹道具
- **获取方式**：钓起水下的"炸弹"物品后自动存入道具栏
- **持有上限**：3个
- **使用方式**：鱼钩正在收回物品时，点击UI"炸毁"按钮
- **效果**：
  - 立即丢弃当前钩上的物品（不计分）
  - 鱼钩以最快速度收回
  - 消耗1个炸弹
- **使用场景**：
  - 钓到旧靴子等陷阱物品时
  - 钓到鲨鱼但时间不足时
  - 想快速重新抛钩时

### 6.2 道具栏UI
- 显示在画面右上角
- 显示当前持有炸弹数量
- 鱼钩收回物品时按钮高亮可点击

## 7. 游戏流程

### 7.1 整体流程
```
启动游戏
   ↓
登录界面（输入玩家名）──→ 连接服务器
   ↓
主菜单
   ├─ 开始游戏
   ├─ 排行榜
   └─ 退出
   ↓
选择关卡（已通关关卡可重玩）
   ↓
进入关卡
   ↓
关卡游玩（钟摆→释放→抓取→收回→计分）
   ↓
关卡结束
   ├─ 达成目标 → 结算（显示得分+时间奖励）→ 下一关
   └─ 未达成   → 失败界面 → 重试 / 返回菜单
   ↓
通关全部关卡 → 通关祝贺界面 → 上传总分到排行榜
```

### 7.2 单次抛钩流程
```
鱼钩钟摆摆动中
   ↓
玩家点击/按空格
   ↓
鱼钩沿当前角度直线下降
   ↓
┌─ 碰到物品 → 抓住物品 → 沿原路收回
│              ↓
│          收回完成 → 物品入篓 → 计分 → 回到钟摆状态
│
└─ 到达底部未碰到物品 → 空手收回 → 回到钟摆状态
```

### 7.3 关卡结算流程
```
时间耗尽 or 手动结束
   ↓
计算物品总分
   ↓
判断总分 ≥ 目标金额？
   ├─ 是 → 计算时间奖励 → 显示通关结算 → 上报关卡结果到服务器
   └─ 否 → 显示失败结算 → 可重试
```
