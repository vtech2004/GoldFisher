# 黄金钓鱼 - 美术资源需求列表

## 1. 概述

本项目为"黄金矿工玩法换皮钓鱼"游戏，所有美术资源采用程序化生成的 2D 卡通像素风格，透明背景 PNG 格式。资源生成脚本位于 `FishApp/Assets/ArtResources/generate_art.py`，生成的图片存放在 `FishApp/Assets/ArtResources/` 对应子目录。

### 1.1 美术风格
- **整体风格**：卡通可爱风，色彩明快，轮廓清晰
- **配色方案**：海洋蓝绿色调为主，物品用鲜艳色彩突出
- **格式**：PNG（RGBA，支持透明背景）
- **尺寸规范**：
  - 背景：1024×1024（可平铺/拉伸）
  - 角色/物品：128×128
  - UI 图标：64×64
  - UI 按钮：256×80
  - 面板：512×512（可九宫格切片）

## 9. 生成方式

所有资源由 Python 脚本 `FishApp/Assets/ArtResources/generate_art.py` 使用 Pillow 库程序化生成。

```powershell
# 生成全部资源
cd e:\Miracle\GoldFisher\FishApp\Assets\ArtResources
python generate_art.py
```

生成后可在 Unity 中导入 `ArtResources` 文件夹，设置 Sprite 导入参数后拖入场景使用。
